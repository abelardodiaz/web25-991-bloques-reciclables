# 991 vs 996: Comparativa

> Documento consultable para entender la diferencia entre ambos proyectos.

---

## Tabla Comparativa

| Aspecto | 996 (Agentes Demo) | 991 (Bloques Reciclables) |
|---------|--------------------|-----------------------|
| Proposito | Orquestar, documentar, preparar proyectos | Contener bloques de codigo reutilizable |
| Contenido | Preparativos, templates, agents, commands | Packages Python, packages TS, templates Copier |
| Quien lo usa | Claude 996 al crear proyectos | Cualquier proyecto, en cualquier momento |
| Open source | No (tiene API keys, config interna) | Si, 100% publico |
| Ubicacion | `prweb/` (privado) | `prweb/public/` (publico) |
| Tipo | Documentacion + orquestacion | Library / Monorepo |
| Versionado | Semantico por docs | Semantico por packages |
| Dependencias | Ninguna externa | uv, pnpm, Copier, Turborepo |
| Repo | GitLab + GitHub (privado) | GitLab + GitHub (publico) |
| Server destino | Local Windows | Local Windows (`prweb/public/`) |

---

## Relacion

996 **prepara** proyectos. 991 **provee bloques** que esos proyectos consumen.

```
996 (privado)                     991 (publico)
+--------------------+            +--------------------+
| Preparativos       |            | bloque-core        |
| Templates agentes  |  refiere   | bloque-auth        |
| PROMPT-ARRANQUE    | ---------> | bloque-multitenant |
| Commands           |            | bloque-ui          |
| API keys           |            | templates Copier   |
+--------------------+            +--------------------+
        |                                  ^
        |  crea proyecto hijo              |
        v                                  |
+--------------------+                     |
| Proyecto hijo      |  instala bloques    |
| (ej: web25-056)    | --------------------+
+--------------------+
```

---

## Regla Simple

- Si necesitas **crear un proyecto nuevo** -> habla con 996
- Si necesitas **un bloque de codigo** -> instala desde 991
- Si necesitas **ambos** -> 996 te dice cuales bloques de 991 usar
